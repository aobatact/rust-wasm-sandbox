フルスクラッチでwasmしたい

準備(Optional)
```
cargo install --git 'https://github.com/alexcrichton/wasm-gc'
```

ビルド
```
cargo build --target=wasm32-unknown-unknown --release
wasm-gc .\target\wasm32-unknown-unknown\release\rust-mywasm.wasm -o .\static\index.wasm
```

簡易サーバー
```
cargo run -p server
```
